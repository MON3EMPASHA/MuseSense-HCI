MuseSense: Tangible + Gesture + Face Interactive Museum Guide (HCI Project) 
1) Project Summary 
MuseSense is an interactive museum experience where visitors explore exhibits using 
TUIO tangible markers (physical cards/tokens placed on a table), hand gestures (to 
curate lists and control views), and facial expression analysis (for login + emotion
aware personalization). 
Visitors can: 
пВЈ Place a tangible marker representing an artifact/exhibit on an interactive surface 
to instantly view it in 2D/3D with media, context, and recommendations. 
пВЈ Use hand gestures to quickly save items into personal lists: Favorites, Explore 
Later, and Good to See. 
пВЈ Log in / register using face-based identity verification, and optionally enable 
emotion-aware personalization where the system uses their facial expression 
(e.g., happiness) and perceived engagement to recommend categories. 
Goal: Improve engagement, reduce friction in navigation, and make exploration playful 
and accessible using multi-modal HCI interaction.
